# gGui: gPhoton glue user interface.

## Prerequisits: 
* pipenv
* Python 3.7.1 64-bit

## Installation Instructions:
Inside the cloned directory:
```console
pipenv install
```
## Sample Usage
1. Activate your gGui pipenv environment:
```console
pipenv shell
```
2. Use the included CLI startup script:
```console
python ./gguicli_startup.py
```
